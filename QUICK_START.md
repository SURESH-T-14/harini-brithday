# 🎉 QUICK START GUIDE

## Installation (30 seconds)

1. **Open `index.html`** in your web browser (double-click or right-click → Open with Browser)
2. **That's it!** The website will work immediately

## Add Your Content (5 minutes)

### 1. Add 15 Memory Photos
- Save 15 photos as: `memory1.jpg`, `memory2.jpg`, ... `memory15.jpg`
- Place them in: `assets/images/`
- Size: Around 500x500px each

### 2. Add Background Music
- Save your music as: `background.mp3`
- Place it in: `assets/music/`
- Suggestion: Soft, emotional, instrumental

### 3. Add Memory Video (Optional)
- Save as: `memory.mp4`
- Place in: `assets/videos/`
- Length: 2-5 minutes recommended

### 4. Customize the Letter
- Open `script.js`
- Find: `const birthdayLetter = "..."`
- Replace with your personal message

### 5. Update Photo Captions
- Open `script.js`
- Find: `const balloonCaptions = [...] `
- Update with 15 captions (one for each photo)

## Test & Send

1. Open `index.html` in browser
2. Click "START" button
3. Follow the experience
4. Test on mobile (very important!)
5. Send the link to your friend!

## How the Experience Works

```
1. User clicks START
   ↓
2. 1-minute countdown to midnight
   ↓
3. Cake animation (candle blows out)
   ↓
4. Pop 15 balloons to reveal memory photos
   ↓
5. Watch memory video
   ↓
6. Read heartfelt letter
   ↓
7. Final celebration scene!
```

## Tips for Best Results

🎯 **Photos**
- High quality, well-lit photos
- Mix of group photos and candid moments
- Varied photo types (adventures, funny moments, sweet memories)

🎵 **Music**
- Soft, emotional background music
- Instrumental or minimal vocals
- Avoid copyrighted music (use royalty-free)

💌 **Letter**
- Write from the heart
- Include specific memories
- Keep it between 200-400 words
- End with warm wishes

📸 **Captions**
- One caption per photo
- Use emojis for fun
- Example: "Road trip shenanigans 🚗"

## Common Fixes

### Photos not showing?
- Check file names: `memory1.jpg`, `memory2.jpg`, etc.
- Check path: `assets/images/`
- Check format: JPG, PNG, or WebP

### Music not playing?
- File name: `background.mp3`
- Path: `assets/music/`
- Try clicking anywhere on page first (browser autoplay policy)

### Video not playing?
- File name: `memory.mp4`
- Path: `assets/videos/`
- Format: MP4 (H.264 codec)
