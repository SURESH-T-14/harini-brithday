# Birthday Surprise Website - Assets Guide

## Images (15 Memory Photos)

Place your 15 favorite memory photos in this folder with names:
- `memory1.jpg` through `memory15.jpg`

Each photo should be:
- **Format**: JPG, PNG, or WebP
- **Size**: Recommended 400x400px to 600x600px
- **Quality**: Optimized for web (50-100KB each)

### Suggested Photos
1. memory1.jpg - First day you met
2. memory2.jpg - Funny moment
3. memory3.jpg - Adventure together
4. memory4.jpg - Sweet moment
5. memory5.jpg - Group photo
... and so on

## Notes
- Photos will appear when balloons are popped
- Each photo gets a random caption from the predefined list
- Higher resolution images = better quality
- Optimize for mobile performance

## Placeholder Setup
If you want to test the website with placeholder images, you can:
1. Use any 15 JPG images
2. Or use a placeholder service like:
   - https://via.placeholder.com/500x500?text=Memory+Photo
   - https://picsum.photos/500/500?random
