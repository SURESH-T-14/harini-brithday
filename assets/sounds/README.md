# Birthday Surprise Website - Sound Effects Guide

## Sound Effects Files

The following sound effects are used throughout the experience:
(Currently the sound system is simulated in the code)

### Recommended Sound Files
To add actual sound effects, create these files in this folder:

1. **countdown.mp3** - Subtle ticking sound for countdown
2. **celebration.mp3** - Cheer/celebration sound when timer ends
3. **pop.mp3** - Balloon pop sound (small, quick)
4. **confetti.mp3** - Confetti/party horn sound
5. **balloons.mp3** - Whoosh/balloon float sound
6. **candle-blow.mp3** - Whoosh/blow sound for candle
7. **transition.mp3** - Smooth transition sound
8. **letter.mp3** - Soft, emotional sound for letter
9. **final.mp3** - Happy, celebratory final scene sound

### Requirements
- **Format**: MP3 (recommended)
- **Quality**: 128kbps or higher
- **Duration**: Keep short (0.5-2 seconds for effects)
- **Size**: Individual files under 200KB
