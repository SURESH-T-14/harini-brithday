# Birthday Surprise Website - Music Guide

## Background Music

Place your background music file in this folder as:
- `background.mp3`

### Requirements
- **Format**: MP3 (recommended) or WAV/OGG
- **Duration**: Minimum 2 minutes (loops throughout experience)
- **Quality**: 128kbps or higher
- **Mood**: Soft, emotional, romantic/friendship vibes

### Suggestions
- Cinematic instrumental music
- Piano compositions
- Ambient music
- Soft background score
- Emotional soundtracks

### Royalty-Free Music Sources
- https://www.bensound.com/
- https://incompetech.com/
- https://www.zapsplat.com/
- https://www.freepik.com/
- YouTube Audio Library

## Notes
- Music will loop throughout the experience
- User can mute/unmute with the music button
- Background music starts when user clicks START
- Keep file size under 5MB for optimal performance
