# Birthday Surprise Website - Video Guide

## Memory Video

Place your memory video file in this folder as:
- `memory.mp4`

### Requirements
- **Format**: MP4 (H.264 codec)
- **Duration**: 2-5 minutes recommended
- **Resolution**: 1920x1080 or lower
- **Bitrate**: 2-5 Mbps for quality
- **File Size**: Keep under 50MB
- **Frame Rate**: 24-30 fps

### Suggestions
- Compilation of best moments
- Slideshow of photos with music
- Video clips of memories
- Montage of fun times together
- Home video memories
- Vacation/adventure footage

### Professional Video Creation Tools
- **Adobe Premiere Pro** - Professional editing
- **DaVinci Resolve** - Free, powerful editor
- **CapCut** - Mobile-friendly editor
- **iMovie** - Mac (built-in)
- **Windows Photos** - Windows (built-in)
- **Shotcut** - Free, open-source
- **OBS Studio** - Screen recording

### Video Conversion Tools
If you have videos in other formats, convert to MP4:
- https://www.convertio.co/
- https://online-convert.com/
- FFmpeg (command-line tool)
- VLC Media Player

## Notes
- Longer videos = larger file size (slower to load)
- Higher resolution = better quality but larger file
- Include music/audio for emotional impact
- Test video plays on different browsers
- MP4 format ensures compatibility
- Keep file under 50MB for optimal performance

## Tips for Best Results

### Photo Slideshow
- 3-5 seconds per photo
- Add transitions between photos
- Include background music
- Total: 2-3 minutes

### Video Montage
- Mix different videos
- Add transitions and effects
- Keep music consistent
- Tell a story

### Quality Settings
- Video: H.264 codec
- Audio: AAC or MP3
- Resolution: 1920x1080 or less
- Bitrate: 3-5 Mbps
